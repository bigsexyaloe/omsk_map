# Omsk Map — Интерактивная карта: Заводы Омска времён Великой Отечественной войны

Веб-сайт с интерактивной картой (Leaflet + OpenStreetMap), на которой отмечены ключевые промышленные предприятия Омска, эвакуированные в город в годы Великой Отечественной войны. При нажатии на метку открывается окно с фотографией и описанием завода.

## Основные характеристики

- **Аудитория:** 50–100 уникальных посетителей в день
- **Характер данных:** статичные (точки и описания меняются крайне редко)
- **Разработка:** AI-assisted (человек — направление и ревью кода)
- **Архитектура:** лёгкая микросервисная

## Технологический стек

| Компонент      | Технология                    |
|----------------|-------------------------------|
| Frontend       | HTML + CSS + JavaScript + Leaflet + OpenStreetMap |
| Backend API    | Python FastAPI                |
| Данные         | JSON-файл                     |
| Контейнеризация| Docker + Docker Compose       |
| Веб-сервер     | Nginx (раздача статики + прокси на API) |

## Быстрый старт

```bash
docker-compose up --build
```

После запуска сайт будет доступен по адресу: `http://localhost:8080`

## Разработка

### Запуск без Docker

**Backend:**
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

**Frontend:**
Достаточно открыть `frontend/index.html` в браузере или использовать любой статический сервер:
```bash
cd frontend
python -m http.server 3000
```
